VisionLive TestFramework Help

Key words

LOG text - will add "Message" type log with description text to Log.
Example: LOG some_text
Warning: text must not contain spaces.
STOP - will end the test.
Example: IF BLC_sensorBlack0 != BLC_sensorBlack0_sb THEN STOP
CLICK object - will emit click signal of object .
Example: CLICK Exposure_auto_cb
Warning: CLICK applies to QPushButton, QRadioButton, QCheckBox, QAction object types.
OBJEXIST object - checks if object exists. Usefull for checking if some dynamically created objects has been created.
Example: OBJEXIST Exposure_auto_cb
FILEEXIST path - checks if specyfied file exists. Usefull for checking if files were created after using exportObjectList() or saveConfiguration().
Example: FILEEXIST C:\Work\234.jpg
WAIT time - pauses test for time miliseconds.
Example: WAIT 5000
SET object , param - sets object value to param. param can be another object.
Example: SET BLC_sensorBlackLevel0_sb 22
Example: SET BLC_sensorBlackLevel0_sb BLC_sensorBlackLevel1_sb
Warning: Value type hold by object and param must be the same (int, double, string).
CALL functionName() param1 param2 (if required) - request execution of specyfied function with provided parameters.
Example: CALL compareParameters()
Example: CALL saveLog() C:/log.txt
param1 <, >, <=, >=, ==, != param2 - compares param1 and param2 using provided operator.
Example: BLC_sensorBlackLevel0_sb == 22
Example: BLC_sensorBlackLevel0_sb <= BLC_sensorBlackLevel0
Warning: Value types of param1 and param2 must be the same (int, double, string).
IF, THEN, ELSE - provides conditional execution of described above commands.
Example: If BLC_sensorBlackLevel0_sb == 22 THEN SET BLC_sensorBlackLevel0_sb 23 ELSE STOP
Warning: Stacing of "IF", "THEN" ane "ELSE" statements is not supported.

Supported functions

exportObjectList() path - saves object list to xml file.
saveLog() path - saves logs to txt file.
compareParameters() - requests parameter list from ISP and compares it with VisinLive parameter list. If comparison failes test ends.
Info: Function goes through all parameters from VisionLive parameter list and if the parameter also exist on ISP parameter list then values of parameters are compared.
connect() port - starts connection on port port .
disconnect() - stops connection.
isDisconnect() - Checks if VL is connected to ISPC_tcp. Usefull to test if some actions don't cause ISPC_tcp to terminate (which may be the cause of connection lost).
Example: IF CALL isDisconnect() THEN STOP
applyConfiguration() - applyies parameters configuration to ISP.
loadConfiguration() path - loads parameters configuration from .txt file.
saveConfiguration() path - saves current parameters configuration to .txt file.
triggerAction() action - emits trigger signal on specyfied QAction object (equivalent to CLICK ).
record() captureName numFrames - requests recording of numFrames frames under captureName name.
Info: After succesfull execution new object (of name captureName _thumbnail) will be created.
saveCaptureAsFLX() thumbnail file - requests saving capture accessed by thumbnail object into file file.
Warning: When saving as FLX file extension must be ".yuv" (for YUV image) or ".flx" (for others).
saveCaptureAsJPEG() thumbnail file - requests saving capture accessed by thumbnail object into file file.
Warning: When saving as JPEG file extension must be ".jpg".
