Detailed Description

LightCorrection is an auxiliary class employed to store a given configuration of brightness, saturation, contrast and sharpness and to provide some methods and operators to easy the interpolation between different configurations for the ControlLBC class.

Each configuration is associated with a given light level which is an estimation of the amount of light captured by the sensor.
