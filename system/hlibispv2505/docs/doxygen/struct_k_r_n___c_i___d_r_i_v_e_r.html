IMG_UINT32 KRN_CI_DRIVER::uiTilingStride

If non 0 will be used to enforce that all tiled buffers have this stride.

If computations for tiled buffers needs bigger tiled buffers then configuration will fail.

Warning
tiling stride has to be a power of 2
