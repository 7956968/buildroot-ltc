Detailed Description

Implementation of Platform allocator for carved out memory.

Implementation of PCI carved out memory for IMG on board PCI memory.

There is information about the device address and it can use the MMU heap to choose where to allocate in the reserved memory.

Can be used as an example on carved out memory allocation.

Note
No changes should be needed if PAGE_SIZE >= device MMU page size but untested
