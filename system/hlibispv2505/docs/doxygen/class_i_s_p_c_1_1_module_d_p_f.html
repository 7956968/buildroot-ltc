Detailed Description

Defective Pixel Fixing (DPG) HW configuration.

Setup function configures the MC_DPF elements of the MC_PIPELINE attached to the owner Pipeline.

ModuleDPF is part of both Setup Modules and Setup Modules (statistics) because it corrects defects (by detection or input map) as well as generate an output map of defects which can be considered as statistics.

Output map information is generated in Metadata::defectiveStats and content in Shot::DPF.
