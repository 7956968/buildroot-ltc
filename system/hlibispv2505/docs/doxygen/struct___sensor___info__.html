IMG_BOOL _Sensor_Info_::bBackFacing

Is this a back or front facing camera.

To be used by application layer to determine horizontal/vertical flip orientations

enum MOSAICType _Sensor_Info_::eBayerEnabled

enabled CFA filter layout (including flipping)

This is filled up by sensor's function as sensors may have different patterns if flipping is enabled or not

Should be derived from SENSOR_INFO::eBayerOriginal and SENSOR_STATUS::ui8Flipping

enum MOSAICType _Sensor_Info_::eBayerOriginal

original CFA filter layout (before flipping)

eBayerEnabled for currently enabled bayer format

Note
SHOULD NOT be affected by mode selection
