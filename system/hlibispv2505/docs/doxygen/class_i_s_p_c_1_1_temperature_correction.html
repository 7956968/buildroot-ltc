Detailed Description

Provides functionality to calculate correlated colour temperatures and provided interpolated colour correction transforms corresponding to a requested temperature.

The collection of possible corrections can be defined by defaults or loaded from a ParameterList. There is functionality for automatic interpolation of corrections if a requested temperature doesn't exactly match one of the stored transforms.
