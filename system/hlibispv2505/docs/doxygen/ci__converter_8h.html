Typedef Documentation

typedef struct CI_CONV_MIPI_PRIV CI_CONV_MIPI_PRIV

When configuring a converter to use MIPI this will be dynamically allocated and attached to the converter.

User can change the virtual channel and the frame number that will be encoded in the mipi packets.

The CI_ConverterConvertFrame() should update the line number.

typedef struct CI_CONVERTER CI_CONVERTER

Converter object that will convert frames into data-generator formats.

Can be memset to 0 as an initialisation.

IMG_UINT32 CI_ConverterFrameSize ( const CI_CONVERTER *  pConverter,
IMG_UINT32  ui32Width,
IMG_UINT32  ui32Height 
)

Returns the encoded frame size including headers and footers.

If used with ui32Height = 1 will return the biggest possible stride (it is assumed that the format cannot have both header AND footer on 1 stride).

Stride is rounded up to the system alignment and then multiplied by ui32Height to give the size

Parameters
[in]pConverter
[in]ui32Widthin pixels
[in]ui32Heightin lines - give 1 to compute the stride
