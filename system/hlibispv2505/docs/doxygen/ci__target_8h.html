Variable Documentation

TAL_sMemSpaceInfo gasMemSpaceInfo[]
static

The definition of the memory space of the device.

Memory space definition:

  1. Memory space name.
  2. Memory space type:
    • if register: TAL_MEMSPACE_REGISTER
    • if memory: TAL_MEMSPACE_MEMORY
  3. Memory space info struct:
    • if register:
      1. Base address of device registers
      2. Size of device register block
      3. The interrupt number. If no interrupt number: TAL_NO_IRQ
    • if memory:
      1. Base address of memory region
      2. Size of memory region
      3. Memory guard band
