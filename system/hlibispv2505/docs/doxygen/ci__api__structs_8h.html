typedef struct CI_BUFFID CI_BUFFID

Buffer ID structure - 1 ID per importable buffer.

Used to simplify API calls that would change when additional IDs are needed

When using ION these are the ION IDs

Warning
Strides CANNOT be changed if the buffer is tiled (HW limitation). If the buffer is tiled ensure a stride of 0 is given.
