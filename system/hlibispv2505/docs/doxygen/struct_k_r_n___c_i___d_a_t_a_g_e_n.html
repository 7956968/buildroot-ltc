IMG_BOOL8 KRN_CI_DATAGEN::bNeedsReset

Needs reset before submitting next frame.

If HW sends DG_INT_ERROR interrupt the HW needs a reset to be able to capture the next frame

This is written during interrupt context, DG start and DG trigger.

If written cannot push frames until a reset
