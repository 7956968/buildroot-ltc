Detailed Description

The ControlDNS class implements a simple dynamic control for the denoiser (DNS) configuration module, updating the ISO_GAIN parameter of the DNS according to the sensor capture settings.

Uses the Sensor to get information to update ModuleDNS.

IMG_RESULT ISPC::ControlDNS::configureStatistics ( )
protectedvirtual

Configures the needed statistics for the module to perform its update() function properly.

Should configure the statistics of the Pipeline object related to the Camera that owns the control module (as only that one will provide statistics).

Returns
IMG_SUCCESS
IMG_ERROR_UNEXPECTED_STATE if needed modules were not found in Pipeline
IMG_ERROR_NOT_SUPPORTED if module should not be programmed using that function
IMG_ERROR_NOT_INITIALISED if module does not have owner to configure or is badly configured (e.g. needs a sensor pointer)

Does nothing, module does not use statistics
