Detailed Description

When configuring a converter to use MIPI this will be dynamically allocated and attached to the converter.

User can change the virtual channel and the frame number that will be encoded in the mipi packets.

The CI_ConverterConvertFrame() should update the line number.

IMG_SIZE(* CI_CONV_MIPI_PRIV::pfnMipiWrite)(const IMG_UINT16 *inPixels, IMG_UINT32 inStride, IMG_UINT32 NpixelsPerCh, IMG_UINT8 *outPixels)

Similar to CI_CONVERTER::pfnLineConverter but only for MIPI data.

MIPI line conversion has:

  • line header (with optional line start if bLongFormat)
  • line data
  • line footer (CRC + optional line end if bLongFormat)

This function only converts the line data. If NULL is given to outPixels does not write but returns the size.
