Detailed Description

Vibrancy part of the Main Image Enhancer (MIE) HW block.

Setup function configures the vibrancy related elements of MC_MIE from the MC_PIPELINE attached to the owner Pipeline.

The setup also uses the information from ModuleMIE::MIE_BLC

Example of generating a non-filtered saturation curve using 2 points (assuming existance of (0.0;0.0) and (1.0;1.0)

Generate a 3 linear segments curve from (0.0;0.0) to point 1, point 1 to point 2 and point 2 to (1.0;1.0).

If in2 > in1 the values will be swapped.
