Detailed Description

Structure containing the information about a pixel format.

  • 420 PL12 8b (NV12;NV21):{ YUV,2,2 } + { 8,1,1,LSB }
  • 420 PL12 10b (NV12-10;NV21-10): { YUV,2,2 } + { 10,3,4,LSB }
  • 422 PL12 8b (NV16;NV61): { YUV,2,1 } + { 8,1,1,LSB }
  • 422 PL12 10b (NV16-10;NV61-10): { YUV,1,2 } + { 10,3,4,LSB }
  • RGB888-24: { RGB,1,1 } + { 8,1,3,LSB }
  • RGB888-32: { RGB,1,1 } + { 8,1,4,LSB } or BGR888-32
  • RGB101010: { RGB,1,1 } + { 10,1,4,LSB }
