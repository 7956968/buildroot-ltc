Detailed Description

HW module configurations that are not producing statistics outputs.

These are the classes that handle the convertion from high level parameters to low-level pre-register values (to an MC_PIPELINE). The Setup Modules (statistics) group contains additional modules that generate statistics from the processed image.

All classes should derive from SetupModule to be storable in a Pipeline object.

The CameraFactor class should be used to list all needed modules per HW version.
