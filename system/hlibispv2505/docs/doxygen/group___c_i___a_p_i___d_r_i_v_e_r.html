IMG_RESULT CI_DriverGetRTMInfo ( CI_CONNECTION *  pConnection,
CI_RTM_INFO *  pRTM 
)

Get the Real Time Monitoring information from HW.

RTM are core registers that can be used to debug the HW. Also gathers information about the status of the contexts.

Warning
Should not be used to monitor if a context or another could start because it is only monitoring the HW state (a HW context can be acquired by a SW object and still be IDLE until its associated gasket receives data)
Parameters
pConnection
[out]pRTM
Note
The number of read entries in pRTM is available in CI_HWINFO::config_ui8NRTMRegisters

IMG_RESULT CI_DriverVerifLinestore ( CI_CONNECTION *  pConnection,
CI_LINESTORE *  pLinestore 
)

Verify that a linestore respects the basic linestore rules.

Note
Computes the aSize elements from the aStart elements

Linestore rules are:

  • the sum of the sizes are lower than the maximum value
  • each context's linestore does not overlap with another
Warning
Does not check if a linestore is active and can be changed.
Parameters
pConnection
[in,out]pLinestore
Returns
IMG_SUCCESS
IMG_ERROR_UNEXPECTED_STATE if driver is not initialised
IMG_ERROR_INVALID_PARAMETERS if pConnection is NULL
IMG_ERROR_FATAL if context's linestore overlaps
See Also
Delegates to CI_LinestoreComputeSizes
