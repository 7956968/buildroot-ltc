Detailed Description

Memory Colour part of the Main Image Enhancer (MIE) HW configuration.

Setup function configures part of the MC_MIE elements of the MC_PIPELINE attached to the owner Pipeline.

The ModuleVIB configures the vibrancy part of the MIE HW block.

Note
The MIE block in HW has MIE_NUM_MEMCOLOURS (3) memory colours. Each represent a single colour to enhance. There are MIE_NUM_SLICES (4) slices per memory colours in the luma and chroma space.

void ISPC::ModuleMIE::calculateMIEExtentParameters ( const double  aspect,
const double *  extent,
int &  kb,
int &  kr,
double *  scale 
)
static

This function transforms higher level parameters extent and aspect into register values kb, kr, scale.

The high level formula is L = exp(|cb|/sigma_b + |cr|/sigma_r)

The low level (register) formula is L = exp(scale*(Kb*|cb| + Kr*|cr|)

[in]extent[MIE_NUM_SLICES]
