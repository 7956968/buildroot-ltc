IMG_RESULT ISPC::Control::addPipelineToAll ( Pipeline *  pipeline)

Add the Pipeline object to all the registered modules.

May not want to use that function if the control module list contains modules that expect to be run alone (e.g. Auto Exposure).

Will try to add the Pipeline to all modules regardless of previous errors

IMG_RESULT ISPC::Control::registerControlModule ( ControlModule *  module,
Pipeline *  pipeline 
)

Register a control module in the list. It's not possible to register two modules with the same ID.

All the registered module's update function will be invoked when required.

Note
The deletion of the registered modules will be in charge of the control module (so registered modules shouldn't be deleted outside this class once they have been registered).
Parameters
moduleControl module to be registered
pipelinePointer to the Pipeline object the control module is going to interact with
Note
Modules will used the pipeline's Sensor pointer to access the sensor object
Returns
IMG_SUCCESS

IMG_RESULT ISPC::Control::setPipelineOwnerToAll ( Pipeline *  pipeline)

Set the given Pipeline as the owner of all the registered modules.

Warning
the Pipeline needs to be in the list of ALL modules to become the owner. Consider using addPipelineToAll() before calling that function.
Note
Owner is set to all modules when registering them

Will try to set the Pipeline to all modules regardless of previous errors

Returns
IMG_SUCCESS
IMG_ERROR_FATAL if one or more module failed to set the Pipeline
