static IMG_RESULT CI_LinestoreComputeSizes ( const CI_HWINFO *  pHWInfo,
CI_LINESTORE *  pLinestore 
)
static

Compute the sizes of the linestore structure using the start points.

The size of context[i] is computed from its start point up to the start of the next context. If the next starting context is outside of the maximum size for context[i] the maximum single size is used.

It is possible that a non-started context has a size of 0

Parameters
[in]pHWInfoto get the maximum size of the linestore
[in,out]pLinestorestructure to read and update
Returns
IMG_SUCCESS
IMG_ERROR_VALUE_OUT_OF_RANGE if a starting point is after the last possible element (max-1)
IMG_ERROR_FATAL if the sum of the element's size is > max (should not happen as size is computed in this function)
