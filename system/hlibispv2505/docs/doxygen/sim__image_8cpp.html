IMG_RESULT SimImageIn_loadFLX ( sSimImageIn *  pSimImageIn,
const char *  pszFilename 
)

Open a file for reading.

Loads either a bayer or a RGB/RGBA FLX image.

Supports Multi-segment FLX that do not change the format, bitdepth and size per segment. If a multi-segment image changes these it will load only the frames of the 1st segment.

Returns
IMG_SUCCESS
IMG_ERROR_INVALID_PARAMETERS if pSimImageIn or pszFilename is NULL
IMG_ERROR_ALREADY_INITIALISED if loadFromFLX or pBuffer already initialised (call SimImageIn_close() beforehand)
IMG_ERROR_MALLOC_FAILED if internal allocations failed
IMG_ERROR_NOT_SUPPORTED if the given FLX format is not supported
