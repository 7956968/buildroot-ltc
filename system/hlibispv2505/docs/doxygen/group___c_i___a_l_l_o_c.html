IMG_RESULT CI_ALLOC_GetTileInfo ( IMG_UINT32  ui32TiledScheme,
struct CI_TILINGINFO *  pResult 
)

Get tiling information relative to the used tiling scheme.

IMG MMU tiling schemes are:

  • 256B x 16 lines
  • 512B x 8 lines
Parameters
[in]ui32TiledSchemethe width of the tiling scheme in bytes (e.g. 256)
[out]pResultscheme structure to fill - asserts it is not NULL
Returns
IMG_SUCCESS
IMG_ERROR_NOT_SUPPORTED if the tiling scheme is not suported!

Get RGB buffer size information.

Parameters
[in]pTypeformat - it is not checked if it is RGB
Note
HDR insertion only supports BGR_161616_64 and cannot be tiled
Parameters
[in]ui32Widthin pixels of the buffer
[in]ui32Heightin lines of the buffer
[in]pTileInfotiling information - NULL if not used
Warning
TILING is not supported by HDR insertion - will return IMG_ERROR_NOT_SUPPORTED if pTileInfo isn't NULL while using BGR_161616_64!
Parameters
[out]pResultoutput information about the buffer
Returns
IMG_SUCCESS
IMG_ERROR_NOT_SUPPORTED if selected tiling setup needed is not possible
Note
also works for Bayer (RGGB) formats and HDR extraction (special RGB)
