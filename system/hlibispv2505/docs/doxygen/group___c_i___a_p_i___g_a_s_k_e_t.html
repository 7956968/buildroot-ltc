IMG_RESULT CI_GasketAcquire ( CI_GASKET *  pGasket,
CI_CONNECTION *  pConnection 
)

Acquire the gasket - release with CI_GasketRelease()

If acquired the gasket information is written to registers and started otherwise nothing happens.

Warning
This delegates all the checking to the kernel-side
Returns
IMG_SUCCESS
IMG_ERROR_INVALID_PARAMETERS if pGasket is NULL
IMG_ERROR_UNEXPECTED_STATE if not connected to the kernel-side
IMG_ERROR_COULD_NOT_OBTAIN_RESOURCE if the Gasket is already acquired

If the gasket was acquired with this CI_CONNECTION it is stopped otherwise nothing happens

Returns

IMG_ERROR_UNEXPECTED_STATE if not connected to the kernel-side
IMG_ERROR_COULD_NOT_OBTAIN_RESOURCE if the Gasket was not acquired by this kernel-connection
