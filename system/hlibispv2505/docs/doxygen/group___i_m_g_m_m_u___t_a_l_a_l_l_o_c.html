Detailed Description

The TAL implementation of the MMU allocator.

TAL Allocator structure organisation
Warning
All memory operations on blocks allocated by TALALLOC should be using TALALLOC_ComputeOffset() to get the correct TAL IMG_HANDLE/ offset for that operation.

IMG_HANDLE TALALLOC_ComputeOffset ( IMGMMU_TALAlloc *  pAlloc,
IMG_SIZE *  uiOff 
)

Returns the IMG_HANDLE to use with TAL to read/write at this offset of the allocation.

The return handle/offset should be used with TALREG_ or TALMEM_ functions.

For example to read a value from memory:

IMG_HANDLE pCurrMem = NULL;
IMG_SIZE uiOff2 = uiOff;
pCurrMem = TALALLOC_ComputeOffset(pMMUAlloc, &uiOff2);
if (NULL == pCurrMem)
{
return IMG_ERROR_FATAL;
}
ret = TALMEM_ReadWord32(pCurrMem, uiOff2, pui32Value);
Parameters
pAllocallocation to use - asserts it is not NULL
[in,out]uiOffthe current offset value (and output the result offset) - asserts it is not NULL
Returns
the structure pointer or NULL on error

IMGMMU_TALAlloc* TALALLOC_Malloc ( IMG_HANDLE  sTalMemHandle,
IMG_SIZE  uiSize,
IMG_RESULT *  pResult 
)

Allocate some device memory using the TAL Memory interface.

Will allocate as many blocks of IMGMMU_PAGE_SIZE using TAL MEM as needed. A "host" linear memory will also be allocated to allow direct modification of the memory.

Warning
The allocated host linear memory is not memset to any value (it is using IMG_MALLOC)

It is a non Unified Memory Access system, therefore memory should be updated:

  • using TALALLOC_UpdateHost() to update from device before read access
  • using TALALLOC_UpdateDevice() to update the device before write access
Parameters
sTalMemHandleTAL Memory Register bank - asserts that it is not NULL
uiSizein bytes to allocate (will be round up to match the number of pages) - asserts that it is not NULL
pResultresult output - asserts that it is not NULL
Returns
The pointer to the IMGMMU_TALAlloc structure and pResult to IMG_SUCCESS
NULL in case of error and pResult contains:
  • IMG_ERROR_INVALID_PARAMETERS if the allocation failed to round up the size
  • IMG_ERROR_MALLOC_FAILED if the structure allocation failed (including its components and the TAL host linear equivalent of the memory to allocate)
  • IMG_ERROR_FATAL if TALMEM failed to allocate one of the needed page
