Detailed Description

Implementation of the Camera that uses the data generator sensor.

This object will only work if the External Data Generator was compiled in. Use supportDG() to know at run-time if it is available.

Retrieve a previously enqueued shot. Shots must be released afterwards.

If it is specified, it blocks or/and the loop control algorithms are updated.

Parameters
shotShot structure where the capture shot is returned
blockIf true then blocks till timeout
updateControlSpecify if the control algorithms should be updated or not.
Returns
IMG_SUCCESS
IMG_ERROR_UNEXPECTED_STATE if the Camera is in a wrong state
IMG_ERROR_FATAL if the Pipeline object is not created
IMG_ERROR_COULD_NOT_OBTAIN_RESOURCE if block==false and \ no frame available
Delegates to Pipeline::acquireShot() and Control::runControlModules()
