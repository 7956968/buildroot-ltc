Detailed Description

Information about a buffer.

Can be used to retrieve information about buffers from CI user space

Field Documentation

IMG_BOOL8 CI_BUFFER_INFO::bAvailable

is the buffer available for capture

If it is an output buffer available if not already given to HW and not already acquired.

If it is an HDR input buffer is available if not reserved.

If it is an LSH matrix it is available if not currently acquired by the user.
