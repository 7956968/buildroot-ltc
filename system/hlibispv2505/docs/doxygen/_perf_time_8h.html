Detailed Description

Performance of functions.

LOG_PERF_IN log current time for call graph. Absolute time is stored. Save it to be subtracted when LOG_PERF_OUT is called. To allow nested logs create a LIFO. Function name and time is stored.

When LOG_PERF_OUT is called the list is searched for matching function name. Every entry which does not match current function name is removed.
