Detailed Description

Raw Look Up Table (RLT) HW configuration.

Setup function configures the MC_RLT elements of the MC_PIPELINE attached to the owner Pipeline.

IMG_UINT16 ISPC::ModuleRLT::aLUTPoints[RLT_SLICE_N][RLT_SLICE_N_POINTS]

LUT points - meaning varies according to selected mode.

  • if linear interpolation aLUTPoints_X should be considered as 4 segments of points allowing the configuration of a curve with different knee points location.
  • if cubic interpolation aLUTPoints_X should be considered as 1 curve per colour channel
